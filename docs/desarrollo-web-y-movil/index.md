# Desarrollo Web y Móvil

Fundamentos, arquitectura y buenas prácticas para crear aplicaciones modernas, funcionales y profesionales para la web y dispositivos móviles, usando enfoques y herramientas reconocidos en la industria.

La categoría cubre desde el diseño de servicios web tipo API REST y la construcción de Web APIs en .NET Core, hasta el análisis estático con SonarQube y la modernización de sistemas legacy.
