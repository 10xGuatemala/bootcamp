# Introducción a la Inteligencia de Negocio

En este curso aprenderemos los conceptos fundamentales de la inteligencia de negocio (Business Intelligence, BI), enfocándonos en cómo las empresas pueden usar los datos para impulsar decisiones estratégicas. A lo largo del curso, exploraremos los métodos de análisis de datos, el uso de almacenes de datos, y cómo estos elementos se combinan para ofrecer una ventaja competitiva.
